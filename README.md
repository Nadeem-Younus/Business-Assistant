# Business Assistant
## What It Does
	• Write professional emails with research (EnhancedEmailWriter)
        - Email purpose (string)
        - Recipient type (client/team/stakeholder/supplier)
        - Tone (formal/professional/friendly)
    • Generate business reports from data (EnhancedSummarizer)
        - Report type (sales/revenue/performance/quarterly)
        - Data (JSON list of numbers or metrics)
        - Time period (Q1/Q2/Monthly/etc.)
    • Analyze sales/revenue data (Data_Analyze)
        - Sales data (revenue, units sold)
        - Financial data (expenses, profit)
        - Performance metrics (conversion rates, KPIs)
        - Time-series data (monthly, quarterly)
    • Create meeting summaries with action item (EnhancedSummarizer)
        - Raw meeting notes (multi-line text)
        - Meeting date
        - Attendees (optional)
    • Draft client communications
        - Project proposal
        - Status update
        - Response to inquiry
        - Follow-up message
